## üí° Pro Tips from the Industry

### Choosing a Vendor:
1. **Order samples first** - Always see physical quality
2. **Read reviews** - Check BoardGameGeek forums
3. **Get multiple quotes** - Compare at least 3 vendors
4. **Plan for extra time** - Add 2-4 weeks buffer
5. **Budget 10-15% extra** - For damaged/replacement units

### Working with Vendors:
1. **Communicate clearly** - Detailed specs prevent mistakes
2. **Ask questions** - No question is too basic
3. **Request proofs** - Physical proof worth the cost
4. **Build relationships** - Repeat customers get better service
5. **Be flexible** - Some compromises save big money

### Cost Optimization:
1. **Standard sizes** - Custom sizes cost 30-50% more
2. **Bulk ordering** - Double order often only +40% cost
3. **Simple backs** - Same back for all cards saves money
4. **Standard finishes** - Linen is cheaper than plastic-coated
5. **Timing** - Off-peak seasons (Jan-Mar) sometimes cheaper

### Quality Control:
1. **Request samples** - From similar projects
2. **Check color profiles** - CMYK vs RGB differences
3. **Test durability** - Shuffle test prototypes
4. **Verify bleeds** - 1/8" bleed on all artwork
5. **Proof everything** - Typos are permanent!
